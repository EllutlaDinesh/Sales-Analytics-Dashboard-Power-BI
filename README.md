Sales Analytics Dashboard – Power BI

Project Overview

This project focuses on analyzing sales performance using an interactive Power BI dashboard. The objective is to transform raw sales data into meaningful insights that help understand revenue trends, cost behavior, and product performance across regions.

Business Problem

Organizations need a clear view of sales, profit, and discount trends to make informed decisions. Raw data often lacks structure and consistency, making analysis difficult without proper cleaning and transformation.

Data Preparation (ETL)

    1. Cleaned and structured raw sales data using Excel
    2. Handled missing values and standardized fields
    3. Prepared datasets for accurate reporting before loading into Power BI

Dashboard Features

KPI cards for Sales, Profit, Cost, Gross Sales, and Discounts
Country-wise and product-wise sales analysis
Monthly and trend-based visualizations
Interactive slicers and filters for dynamic analysis

Tools Used

Excel (Data Cleaning & ETL)
Power BI (Data Modeling & Visualization)

Key Insights

    1. Identified top-performing products and high-revenue regions
    2. Observed monthly sales trends and discount impact
    3. Enabled easy comparison of performance across categories

Conclusion

This dashboard provides a clear, interactive view of sales performance and supports data-driven business decision-making through effective visualization and analysis.
